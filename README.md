# 🏢 Marsha Rentals - Cloud Property Management

**Marsha Rentals** is a high-fidelity Property Management SaaS platform designed to streamline the rental experience for **Administrators**, **Landlords**, and **Tenants**.

### 🌟 Key Features
- **🏡 For Tenants**: Interactive property search, maintenance requests, secure payments (MPesa/Card), and cloud document storage.
- **📈 For Landlords**: Property listing analytics, tenant management, income tracking, and real-time messaging.
- **🛠️ For Admins**: Full Content Management System (CMS), user administration, platform-wide analytics, and maintenance operations.

### 🚀 Tech Stack
- **Frontend**: HTML5, Vanilla CSS3 (Custom Design System), JavaScript (ES6+).
- **Backend/Database**: [Supabase](https://supabase.com/) (PostgreSQL, Auth, Storage).
- **Icons**: Lucide/Custom SVGs.

### 🛡️ Setup
This project is connected to a live Supabase instance. To run locally:
1. Ensure the `supabaseUrl` and `supabaseKey` in `index.html` are correctly configured.
2. Visit the website to experience the seamless dashboard transitions.
